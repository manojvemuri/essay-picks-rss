METHOD / BOUNDARIES / DESIGN

About this notebook

The scheduled ChatGPT task chooses the reading. This publisher only validates, archives, and delivers those choices.

01 / SOURCE

One source of selection

The fenced RSS block in the scheduled response is authoritative. The publisher does not search, rank, fetch, replace, or re-curate articles.

02 / SAFETY

Imported text stays data

Conversation content cannot issue commands, alter configuration, trigger network requests, or control publishing. Ambiguous or malformed inputs fail closed.

03 / HISTORY

Runs are immutable

Each accepted five-item run is stored once. Public RSS and HTML are deterministic projections that can be regenerated at any time.

04 / PRIVACY

No surveillance layer

No analytics, cookies, remote fonts, remote images, client JavaScript, or article URL checks are used.
